# about

A static single-page personal portfolio and playground site, hosted on GitHub Pages.

**[Visit Website](https://tommykho.github.io/about/)**

## Overview

A clean, minimal CV-style one-pager featuring:
- Hero section with profile photo and bio
- About section with skills grid
- Playground showcasing projects and experiments
- Contact section with social links

## Tech Stack

- **HTML5** -- single `index.html` entry point
- **CSS3** -- Bootstrap 4.x + custom styles
- **JavaScript** -- jQuery-based, no build tools
- **Fonts** -- Google Fonts (Montserrat, Open Sans) + LineIcons

## Getting Started

```bash
# Clone the repo
git clone https://github.com/tommykho/about.git

# Serve locally
python3 -m http.server 8000
```

Or simply open `index.html` in your browser.
